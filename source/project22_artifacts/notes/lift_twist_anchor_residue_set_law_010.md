# Lift & Twist anchor residue set law 010

Status: lift_twist_anchor_residue_set_law_recorded

## Claim

The anchor-side residue set of the four-state local answer-cell target is generated by a two-bit offset law. With shell bit b and rank bit r, the anchor key is 4*b+12*r+12*b*r mod 15. Ordinary states use shared offsets {5,7,12} plus a rank switch; branch states use shared offsets {0,4} plus a branch rank switch. Adding these offsets to the anchor key mod 15 recovers the full observed anchor residue sets for O0,O1,B0,B1.

## Laws

- anchor_key: `a = 4*b + 12*r + 12*b*r mod 15`
- ordinary_offsets: `{5,7,12} union {1+3*(1-r), 2+6*(1-r), 3+6*(1-r)}`
- branch_offsets: `{0,4} union {11-6*r, 13-4*r, 14-4*r} mod 15, plus {12} when r=1`
- anchor_residues: `{anchor_key + offset mod 15 for offset in offsets}`
