\section{Native provenance question}

The native provenance question is:

\[
    \text{Which native G60 relations force the four-state local Lift \& Twist cell?}
\]

The candidate sources include the canonical G60 edge payload, station provenance fields, WXYZTI transition records, and the channel-aware G60-G15 join already matched to the Project 16 twisted G900 candidate.

This paper starts conservatively.  It does not assert that any one source is sufficient.  It asks for a reproducible derivation chain from native records to the local-cell kernel.
